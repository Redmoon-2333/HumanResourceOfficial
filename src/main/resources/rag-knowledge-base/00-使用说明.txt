# 知识库使用说明

本目录用于存放RAG知识库的源文件，支持以下格式：
- .txt（纯文本文件，推荐使用UTF-8编码）
- .docx（Word文档）
- .pdf（PDF文档）

## 目录结构建议

按照以下结构组织知识库文件，便于分类管理和检索。

## 内容编写规范

1. **文件编码**：使用UTF-8编码
2. **标题层级**：使用"一、二、三"或"1. 2. 3."标记
3. **段落分隔**：段落之间使用空行
4. **列表格式**：使用"- "标记无序列表
5. **重点标注**：使用【】标记重要信息

## 初始化步骤

1. 将编写好的文件放入对应分类目录
2. 登录系统，进入"RAG知识库管理"页面
3. 点击"初始化知识库"按钮
4. 等待处理完成，查看初始化报告
