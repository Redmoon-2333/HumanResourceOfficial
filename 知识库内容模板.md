A：只要认真完成任务、积极参与活动，一般都能顺利转正。试用期是相互了解的过程。

Q4：加入后可以中途退出吗？
A：可以，但希望提前沟通。我们理解每个人的情况，会办理正常的离职手续。

Q5：有工资或补贴吗？

  * 提问技巧：STAR原则
  * 候选人评估方法
  * 常见错误与坑
  * 面试礼仪与气场
- 形式：理论讲解 + 模拟面试 + 角色互换

  * 新闻稿件的要素
  * 常用文案模板分享
  * 标题的吸引力技巧
- 形式：模板分享 + 案例分析 + 实战练习
- 成果：每人完成一篇活动总结

- 时长：2小时

  * 可视化工具介绍（如图表秀）
- 形式：工具教学 + 案例实践
- 成果：制作一份数据报告

  * 番茄钟工作法

- 成果：掌握沟通框架和应对策略

二、组织流程

**第一步：主题选择（每月初）**
1. 收集成员需求
   - 发起问卷调查
   - 例会上征求意见
   - 分析当前技能短板

2. 确定月度主题
   - 根据需求排序
   - 考虑时间节点（如招新前讲面试技巧）
   - 平衡硬技能和软技能

3. 预约时间和场地
   - 选定具体日期
   - 预定教室或会议室
   - 确认设备（投影仪、音响）

**第二步：邀请主讲人（提前2周）**
1. 内部主讲人
   - 部长/副部长
   - 有专长的老成员
   - 往届成员

2. 外部嘉宾
   - 其他部门优秀成员
   - 专业老师
   - 企业HR/培训师

3. 沟通确认
   - 说明工作坊目标和参与对象
   - 确定讲授内容和时长
   - 提供必要的资料支持

**第三步：准备工作（提前1周）**
1. 制作宣传物料
   - 设计工作坊海报
   - 编写宣传推文
   - 制作报名表单

2. 准备教学材料
   - 协助主讲人完善PPT
   - 准备案例资料
   - 制作练习题目
   - 准备讲义或资料包

3. 场地和设备
   - 再次确认场地预定
   - 测试投影和音响
   - 准备签到表、名牌、水笔等

**第四步：宣传通知（提前3-5天）**
1. 线上宣传
   - 微信公众号推送
   - 部门群聊通知
   - 朋友圈造势

2. 报名统计
   - 开放报名表单
   - 实时监控报名情况
   - 提醒未报名成员

3. 细节确认
   - 向成员发送提醒通知
   - 明确告知时间、地点、注意事项
   - 建议携带物品（笔记本、笔）

**第五步：现场执行（当天）**
1. 提前到场准备（30分钟）
   - 布置场地和座位
   - 测试设备
   - 准备茶水和签到表

2. 签到与破冰（10分钟）
   - 组织成员签到
   - 发放资料和文具
   - 简单破冰游戏

3. 正式讲授
   - 主持人开场介绍
   - 主讲人授课
   - 现场互动与练习
   - 答疑解惑

4. 总结与反馈
   - 主持人总结要点
   - 发放反馈问卷
   - 合影留念

**第六步：后续跟踪（活动后）**
1. 资料整理与分享
   - 整理课件和笔记
   - 上传到部门共享云盘
   - 微信群分享资源

2. 收集反馈
   - 统计问卷结果
   - 分析满意度和改进建议
   - 总结经验教训

3. 感谢与实践
   - 向主讲人发送感谢信
   - 布置后续练习任务
   - 跟踪成员应用效果

4. 宣传总结
   - 拍摄现场照片
   - 编写活动总结推文
   - 公众号发布

三、质量控制

1. 内容质量
- 与成员实际需求结合
- 首先有理论框架，再有实践案例
- 互动环节充分，避免单向输出
- 提供后续学习资源

2. 主讲人要求
- 专业能力过硬
- 表达清晰，可以调动气氛
- 有实战经验可分享
- 耐心解答问题

3. 时间控制
- 总时长控制在1.5-2小时
- 理论：实践 = 4：6
- 留出足够的互动时间
- 不要超时，影响体验

4. 成果输出
- 每次工作坊都要有具体成果
- 成果可以是作品、计划、笔记等
- 鼓励成员展示和分享
